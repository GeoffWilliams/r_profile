Overview

R_profile::Timezone

Select the active system timezone (requires reboot). Currently supports Linux, Solaris, AIX and Windows. Timezone names for linux need to match the Olsen names available on the current system. On Linux, these are normally found under /usr/share/zoneinfo. On Windows the timezone needs to match a name from tzutil -l

Examples:

Usage

include r_profile::timezone

Setting the timezone for Linux

r_profile::timezone::zone: "Pacific/Honolulu"

Setting the timezone for Windows

r_profile::timezone::zone: "Hawaiian Standard Time"

Parameters:

  • zone (Optional[String]) (defaults to: undef) —

    Timezone to set this node to, eg 'Asia/Hong_Kong'

class r_profile::timezone(
    Optional[String] $zone = undef,
) {

  if $facts['kernel'] == 'windows' {
    class { "windows_timezone":
      tz => $zone,
    }
  } else {
    class { "timezone":
      zone => $zone,
    }
  }
}
